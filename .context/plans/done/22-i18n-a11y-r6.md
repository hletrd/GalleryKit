## 7. Fix passive touch listener preventing scroll suppression (P3)
**Source:** M-13 (R6 review)
**File:** `src/components/photo-navigation.tsx:108-110`
**Confidence:** MEDIUM

`touchmove` registered as `{ passive: true }` — browser can't suppress vertical scroll during diagonal swipe navigation.

**Fix:**
- Make `touchmove` non-passive; call `preventDefault()` when horizontal swipe detected:
  ```ts
  const dx = Math.abs(touch.clientX - startX);
  const dy = Math.abs(touch.clientY - startY);
  if (dx > dy && dx > 10) {
      e.preventDefault(); // suppress vertical scroll during horizontal swipe
  }
  ```

**Verification:**
- [ ] Horizontal swipe navigates photos
- [ ] Vertical scroll works when not swiping horizontally
- [ ] No console warnings about passive listener violations
